Student Management System (SMS) is a desktop-based application developed using Qt Creator and C++ as one of my first major programming projects. The main purpose of this project is to simplify the management of student records through an easy-to-use graphical user interface built with the Qt framework. The application performs complete CRUD (Create, Read, Update, Delete) operations, allowing users to efficiently add new student records, view stored data, update existing information, and remove records whenever required.

The system manages important student details such as Student ID, Name, Date of Birth, Course, and other related information. To ensure persistent data storage, the project uses CSV file handling where all records are stored in a structured comma-separated format. These files can also be opened and viewed in standard text editors like Notepad, making the stored data lightweight, portable, and easy to maintain.

This project was developed using Object-Oriented Programming (OOP) concepts in C++, helping me gain practical experience with classes, objects, encapsulation, and modular programming techniques. In addition, I explored GUI development using Qt Creator, event handling, file management, and user interaction design. Working on this project improved my problem-solving skills and strengthened my understanding of modern C++ application development. It represents an important milestone in my learning journey as a beginner software developer and reflects my interest in building real-world desktop applications using C++ and the Qt framework.
